Основные цели проекта:
-Применить полученные в процессе обучения знания и навыки по работе с данными, алгоритмами машинного обучения, ансамблями;
-Продолжить собирать портфолио на Github. 

Задачи проекта:
- создать модель, которая предсказывает стоимость автомобиля по его характеристикам;
- Принять участие в хакатоне на Kaggle.

Этапы работы над проектом: 
1. Провел разведывательный анализ данных (визуализация, очистка данных, работа с выбросами);
2. Работа с признаками (выбор признаков, генерация новых признаков);
3. Обработка данных для обучения модели;
3. Настройка гиперпараметров, обучение одиночных моделей;
6. Построение ансамблей;
6. Тестирование, проверка точности MAPE;
6. Обоснование выводов и общей логики решения;
7. Принял участие в соревновании на Kaggle;
8. Выгрузил проект для проверки на Github.

В данном проекте углубился в изучение моделей машинного обучения, изучил дополнительные библиотеки и инструменты.
