Основные цели проекта:
-Применить полученные в процессе обучения знания и навыки по работе с данными и основными алгоритмами машинного обучения;
-Продолжить собирать портфолио на Github. 

Задачи проекта:
- Написать скоринговую модель предсказания дефолта клиентов банка;
- Принять участие в хакатоне на Kaggle.

Этапы работы над проектом: 
1. Провел разведывательный анализ данных (визуализация, очистка данных, работа с выбросами);
2. Работа с признаками (выбор признаков, генерация новых признаков);
3. Настроил гиперпараметры;
4. Проверить работу модели и ее точность;
5. Посчитал результат метрики ROC AUC;
6. Обосновал выводы и общую логику решения;
7. Принял участие в соревновании на Kaggle;
8. Выгрузил проект для проверки на Github.

В данном проекте смог применить знания полученные в процессе обучения, от разведывательного анализа данных до запуска и проверки модели обучения.
